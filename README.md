# 🤖 Athena AI Assistant 
A smart conversational AI Assistant built using **Streamlit** and powered by the **Google Gemini API**(`gemini-2.5-flash`).This app tracks chat history and features a clean , user-friendly interface.

## 🚀 Features
* **Real-time AI Chat:** Fast responses using the latest Gemini model.
* **Contextual Memory:** Tracks and displays previous messages in the session.
* **Clear Chat:** Clean up the conversation anytime with a single click from the sidebar.
* **Modern UI:** Built entirely in python using Streamlit components.

## 🛠️ Tech Stack 
* **Language:** Python
* **Framework:** Streamlit
* **AI Model:** Google Gemini API
